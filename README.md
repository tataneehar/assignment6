Assignment Title

GUI-Based Calculator Application Using Python Tkinter

Objective

To develop a simple calculator application using Python’s Tkinter library that performs basic arithmetic operations through a graphical user interface.

Technologies Used

Python 3

Tkinter (GUI Library)

Features

Graphical user interface using Tkinter

Performs basic arithmetic operations:

Addition (+)

Subtraction (−)

Multiplication (*)

Division (/)

Modulo (%)

Clear button to reset input

Equal button to display result

Button-based user interaction

Project Structure
Calculator_Tkinter/
│
├── calculator.py
└── README.md

How to Run the Program

Ensure Python is installed

Open terminal / command prompt

Navigate to the project folder

Run the command:

python calculator.py


Calculator window will appear

Learning Outcomes

Understanding Tkinter widgets

Event handling using command functions

GUI layout management

Basic Python GUI application development
